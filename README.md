## Overview

This project is a web-based platform designed to help users discover nonprofit organizations and volunteer opportunities in the Atlanta area. The website provides information about organizations, available volunteer roles, and a contact interface for user feedback.

## Live Design / Screens

This project includes UI designs created using Canva.

## Key Features

- Homepage encouraging users to volunteer
- Nonprofit organization listings
- Detailed descriptions of organizations and their missions
- Volunteer opportunity listings with requirements
- Contact and feedback form for user interaction

## Example Organizations Included

- Back On My Feet
- Atlanta Community Food Bank
- SafeHouse Outreach

## Volunteer Opportunities

The platform highlights real opportunities such as:
- Community engagement and mentorship
- Food distribution and logistics support
- Outreach and meal service programs

## Project Development Approach

This project was created during a **50-minute time-boxed sprint**, focusing on rapid ideation, design, and execution.

### Sprint Breakdown:
- **Ideation (10–15 minutes):**
  Identified the problem of connecting users to local nonprofits and volunteer opportunities.

- **Design (15–20 minutes):**
  Created UI layouts and structured navigation using Canva.

- **Execution (15–20 minutes):**
  Developed key pages including:
  - Homepage
  - Nonprofit listings
  - Volunteer opportunities
  - Contact/feedback page

This approach emphasized speed, creativity, and practical problem-solving under time constraints.
